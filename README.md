# Lancers Auto Bid

Chrome extension for automatically monitoring and bidding on [Lancers.jp](https://www.lancers.jp) system/web development projects.

See [lancers-auto-bid/README.md](lancers-auto-bid/README.md) for full documentation.

## Quick Start

1. Load `lancers-auto-bid` as an unpacked extension in Chrome
2. Log in to Lancers.jp in the same browser
3. Open the dashboard and configure API keys in Settings
4. Start filtering and bidding from the Status page
